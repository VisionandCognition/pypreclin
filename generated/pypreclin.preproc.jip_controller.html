pypreclin.preproc.jip_controller.call_auto(cmd, lock_file, refresh_winid=True)[source]¶

Execute a JIP command and control the graphical interface.

Parameters:

cmd: list of str

the JIP command that will popup a window.

lock_file: str

the path to a file used to acquired a lock in order to identify properly the window id.

refresh_winid: bool (optional, default True)

if set check that only one JIP alignment is performed at a time. The windows ID is refreshed at each iteration.
Returns:

mi: float

the final optimizer cost function value (mutual information).
