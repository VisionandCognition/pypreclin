Module that contains export utilities.

pypreclin.utils.export.gzip_and_update_outputs(results)[source]¶

Compress file and update the result outputs accordingly. Note that a simplified results structure as produced by the ‘export_interface_results’ function is expected.

Note that the results structure will be updated (ie modified) by the function.

Parameters:

results: dict (mandatory)

a results structure with process name as keys and process results as values. Each process values is expected to contain an ‘outputs’ key containing a dictionary mapping of the process outputs.
pypreclin.utils.export.gzip_file(ungzip_file, prefix='g', outdir=None, remove_original_file=False)[source]¶

Gzip an input file and possibly remove the original file.

Parameters:

ungzip_file: str (mandatory)

an input file to gzip.

prefix: str (optional, default ‘g’)

a prefix that will be concatenated to the produced file basename.

outdir: str (optional, default None)

the destination folder where the Gzip file is saved. If this parameter is None, the input image folder is considered as an output folder.

remove_original_file: bool (optiona, default False)

if True, remove the original file.
Returns:

gzip_file: str

the returned Gzip file.
